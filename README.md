# 🛒 Blinkit Sales Dashboard — Power BI

A comprehensive sales analytics dashboard built in Power BI for **Blinkit — India's Last Minute App**, covering outlet performance, item categories, and sales trends from 2012 to 2022.


---

## 📊 Dashboard Highlights

| KPI | Value |
|-----|-------|
| Total Sales | $1.20M |
| Average Sales | $141 |
| Number of Items | 8,523 |
| Average Rating | 3.9 |

---

## 🔍 Key Insights

- **Outlet Establishment Trend** — Sales peaked at $205K around 2018, with consistent performance across all years
- **Top Item Types by Avg Sales** — Household ($149), Dairy ($148), and Starchy Foods ($147) lead revenue
- **Outlet Size** — Tier 3 outlets generate the highest sales ($472K), followed by Tier 2 ($393K) and Tier 1 ($336K)
- **Outlet Type Performance** — Grocery Stores lead with $151.94K in total sales and 1,083 items
- **FAT Content** — Regular vs Low Fat split tracked across all outlet tiers

## 🧩 Visuals Used

- **KPI Cards** — Total Sales, Avg Sales, No. of Items, Avg Rating
- **Line + Area Chart** — Outlet establishment trend over years
- **Donut Chart** — FAT content split (Regular vs Low Fat)
- **Bar Chart** — Avg sales by item type
- **Donut Chart** — Outlet size distribution
- **Stacked Bar Chart** — Outlet location by tier
- **Matrix Table** — Outlet type breakdown with conditional formatting
- **Slicers** — Outlet Location Type, Outlet Size, Item Type

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **DAX** — for calculated KPIs and measures
- **Power Query** — for data transformation
- **Data Source** — BlinkIT Grocery Data

---

## 🚀 How to Use

1. Clone or download this repository
2. Open `Blinkit.pbix` in **Power BI Desktop**
3. Use the filter panel on the left to slice by:
   - Outlet Location Type
   - Outlet Size
   - Item Type
